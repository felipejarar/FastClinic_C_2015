Campos de datos

int ID
 "ID" contiene el identificador de una vinculacion diagnostico-paciente en la base de datos.
 
int IDPaciente
 "IDPaciente" contiene el identificador de un paciente relacionado con una vinculacion diagnostico-paciente.
 
int IDDiagnostico
 "IDDiagnostico" contiene el identificador de un diagnostico relacionado con una vinculacion diagnostico-paciente.
 
int IDDoctorDiagnostico
 "IDDoctorDiagnostico" contiene el identificador del doctor que llevo a cabo un diagnostico relacionado con una vinculacion diagnostico-paciente.
 
int IDDoctorAlta
 "IDDoctorAlta" contiene el identificador del doctor que dio de alta a un paciente relacionado con una vinculacion diagnostico-paciente.
 
char fechaDiagnostico [DEFAULT_STRING_SIZE]
 "fechaDiagnostico" contiene la fecha en que se llevo a cabo un diagnostico relacionado con una vinculacion diagnostico-paciente. En formato (dd/mm/aa).
 
char estadoDiagnostico [DEFAULT_STRING_SIZE]
 "estadoDiagnostico" contiene el estado del diagnostico relacionado con una vinculacion diagnostico-paciente.
 
char fechaAlta [DEFAULT_STRING_SIZE]
 "fechaAlta" contiene la fecha en que se dio de alta al paciente relacionado con una vinculacion diagnostico-paciente. En formato (dd/mm/aa).
 
char detalleAlta [DEFAULT_LONG_STRING_SIZE]
 "detalleAlta" contiene una breve descripcion del alta de un paciente relacionado con una vinculacion diagnostico-paciente.
